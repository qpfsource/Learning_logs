一、常见文件处理命令
	格式：命令  [-选项] [参数]  例:ls -la /etc
				ls -a（查看所有文件） -l（查看列表，显示属性信息） ls -al [参数]     常见选项
				文件把用户分为：1.所有者（u）2.所属组（g）3.其他人（o）,表示用户和文件之间的关系。（创建文件的人一般为为该文件的所有者）。
				1.1 root（所有者） root（用户组）  1208（文件大小，可以使用-lh查看对应数据） 6月   8 05:24（文件最后修改时间） anaconda-ks.cfg
				2.--rw-r--r-- （代表所有者有读写权限 ，所属组和其他人只有读的相关权限）
					第一个如果是 -表示是一个文件   -d 表示的是一个目录  -l 表示的是一软链接
					u 所有者       g 用户组          o 其 他 人
					  r 读           w 写              x 执行
				3.使用ls -ld查看目录的相关属性
				4.ls -i 查询文件的i节点
二、目录处理命令
	1.创建目录   格式：mkdir -p [目录名] 可以创建多个归属的目录
						mkdir /tmp/test   :在tmp目录下创建test目录
	2.切换目录   格式：cd [目录名]  切换到对应的目录名称
	              pwd :显示当前所在的目录的绝对路径
	3.删除目录    格式:rmdir [目录名] 删除空的目录    例：rmdir /tmp/jpan/boduo（该目录为空）
	
	4.复制目录    格式：cp 需要复制的文件   需要拷贝到的对应目录     cp -rp [源文件目录] [目标文件路径]   -r 复制目录   -p 保留文件属性
	               文件复制：cp /etc/grub.conf /tmp   复制文件到对应的工作目录    可以同时复制多个文件  cp /etc/grub1.conf /etc/grub2.conf /tmp
				   目录复制：cp -r /tmp/jpan /root     复制目录到对应的工作目录，需要加上 -r的选项
				   复制文件并保存文件属性：cp -p /root/install.log /tmp   (复制的同时保持文件的属性)
	5.复制并改名   cp -r /tmp/jpan/install.log /root/shengchao.log (复制目录并更改对应的目录名称) 
	
	6.剪切目录     mv /tmp/jpan/cangjing  /root(将目录剪切到对应的目录)
	               mv /tmp/jpan/longze  /root/nvshen(将目录剪切到对应的目录并改名)    mv 在当前文件下的文件   新的文件名（在文件目录下更改对应的文件的名称）
				   
	7.删除命令   rm [文件名称]    rm test.tst 删除对应的文件      rm -f [文件名称]    rm test.tst 强制删除对应的文件
	             rm -r 删除对应的一个的目录         rm -rf 强制删除对应的一个的目录  
				 
三、文件处理命令
    1.创建文件    touch 文件名称   创建指定名称的对应文件
    
    2.浏览查看文件内容   cat  文件名称    cat /etc/issue          查看文件内容并显示行号   cat -n /etc/issue   (tac  文件名   文件内容倒序显示)

   3.分页查看文件内容      more   文件名称      空格/f 翻页      enter 换行   q 或 Q  退出

   4.分页查看内容       less     /搜索关键字      n 继续向下查看相关内容        可以向上翻页   pageup   向上箭头   向上移动一行

  5.查看文件前几行    head -n 文件名     显示前n行文件内容

  6.查看文件的最后几行   tail -n 文件名称    显示文件最后n行的内容       tail -f 动态显示末尾几行内容

四、链接命令
       ln -s [源文件][目标文件]
            -s 创建软链接
       创建软连接  ：ln -s /etc/issue /tmp/issue.soft   （类似于windows的快捷方式）
       创建硬链接 ： ln /etc/issue /tmp/issue.hard  
      （和源文件内容和大小相同      和cp -p 不同的是建立硬链接后文件内容会同步更新
         删除源文件后硬链接还可以使用     使用ls -i 使用硬链接查看该文件的i节点 ,源文件和硬链接的i 节点内容相同
         硬链接不能做用于目录）
