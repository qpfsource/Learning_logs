Vue-cli3创建项目
	1.创建项目的基本命令:Vue create 项目名       Vue init webpack 项目名称(Vue-cli2创建项目)    
	2.preset:选择配置      pick a preset   （默认选中的babel作用是进行ES6转换ES5，ESlint 为对应的代码结构要求）
	3.选择配置文件的方式，1.dedicated configure (独立的配置文件)    2.package.json(将相关的配置文件放入package.json中)
	4.保存这种项目的结构的相关配置，可以供后期创建项目时使用
	5.选择项目软件包的管理方式  （4之后一般不包含该步骤）
